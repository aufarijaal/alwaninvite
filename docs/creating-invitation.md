# Creating an Invitation

When you click **"Create New Invitation"** from your Dashboard, you'll see a form with several sections. Fill them in from top to bottom. You don't have to complete everything at once — you can save as a draft and come back later.

::: tip
Only fields marked with **\*** (Required) are mandatory. Everything else is optional and can be added or updated at any time.
:::

---

## Section 1 — Basic Information

These fields set up the foundation of your invitation.

### Title _(Required)_

The name of your invitation. This appears at the top of your invitation page.

> Example: _"The Wedding of Budi & Ani"_

### URL Slug _(Required)_

This is the unique web address for your invitation. It looks like:

```
alwaninvite.my.id/invite/your-slug-here
```

It's created automatically from your title, but you can change it. Use only lowercase letters, numbers, and hyphens (no spaces).

> **Tip:** Keep it short and easy to remember — you'll be sharing this link with your guests.

### Language _(Required)_

Choose whether your invitation displays in **English** or **Indonesian (Bahasa Indonesia)**. This changes all text labels on the invitation page (like "RSVP", "Venue", "Date", etc.).

### Theme _(Required)_

The visual design of your invitation. Click **"Choose Theme"** to preview all available themes and pick one that matches your wedding style.

> You can change the theme at any time, even after publishing.

### Publish

Toggle this **on** when you're ready for guests to see your invitation. While it's off (_Draft_), the invitation link won't work for guests — they'll see a blank page.

## Section 2 — Couple Information

This section is filled out for both the **Groom** and the **Bride**. Each person has the same fields:

| Field                 | Required? | Description                                                          |
| --------------------- | --------- | -------------------------------------------------------------------- |
| **Call Name**         | Yes       | The short name shown on the invitation. Usually just the first name. |
| **Full Name**         | Yes       | The complete formal name. Often shown in a smaller font.             |
| **Additional Info 1** | No        | A free-text line. Commonly used for parent names.                    |
| **Additional Info 2** | No        | Another free-text line. Often used for birth order.                  |
| **Additional Info 3** | No        | A third free-text line. Often used for an Instagram handle.          |

> **Example:**
>
> - Call Name: **Budi**
> - Full Name: **Budi Santoso, S.T.**
> - Additional Info 1: **Son of Mr. Hasan & Mrs. Siti**
> - Additional Info 3: **@budi.santoso**

---

## Section 3 — Events

You can add **one or more events** to your invitation (e.g. Akad Nikah + Reception). Click **"Add Event"** to add another.

Each event has these fields:

| Field                | Required? | Description                                                                                                                                                                     |
| -------------------- | --------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Event Type**       | Yes       | Choose: Ceremony, Reception, or Other.                                                                                                                                          |
| **Event Title**      | Yes       | The name of the event. Example: _"Akad Nikah"_, _"Resepsi"_.                                                                                                                    |
| **Start Time**       | Yes       | The date and time the event begins.                                                                                                                                             |
| **End Time**         | No        | When the event ends. Only available for Reception and Other event types. Toggle **"Open end time"** to leave it blank (shows "until done"). Not applicable for Ceremony events. |
| **Location Name**    | Yes       | The name of the venue. Example: _"Ballroom Grand Hyatt Jakarta"_.                                                                                                               |
| **Location Address** | No        | The full address of the venue.                                                                                                                                                  |
| **Map URL**          | No        | A Google Maps link. Guests can tap this to get directions on their phone.                                                                                                       |

---

## Section 4 — Gifts

Let guests know how to send a wedding gift. You can add multiple gift options. Click **"Add Gift"** to add more.

There are four types of gift entries:

| Type               | What to fill in                                                           |
| ------------------ | ------------------------------------------------------------------------- |
| **Bank Transfer**  | Bank name, account holder name, and account number.                       |
| **E-Wallet**       | Provider name (e.g. GoPay, OVO, PayPal) and your registered phone number. |
| **Cryptocurrency** | Coin name and your wallet address.                                        |
| **Gift Registry**  | Store or marketplace name and a link to your wishlist.                    |

---

## Section 5 — Livestream

If you're streaming your event online, you can add a livestream link so guests who can't attend can still watch.

| Field          | Description                                                                                                  |
| -------------- | ------------------------------------------------------------------------------------------------------------ |
| **Platform**   | Where you'll be streaming: YouTube, Instagram Live, Zoom, or Other.                                          |
| **Stream URL** | The direct link to your live stream.                                                                         |
| **Event Date** | The date of the live stream.                                                                                 |
| **Start Time** | What time the broadcast begins.                                                                              |
| **Active**     | Toggle on to show the livestream section on your invitation, or off to hide it without deleting the details. |

---

## Section 6 — Background Music

Add a song that plays automatically when guests open your invitation.

**Where to find music:**

- **System Audio** — Ready-to-use tracks we've included in the app. Browse and preview before selecting.
- **My Uploads** — Your own uploaded songs. Upload them first in the [Media Library](./media-library).

**Options:**

- **Autoplay** — Music starts playing automatically when the invitation opens.
- **Loop** — Music repeats from the beginning when the track ends.

## Section 7 — Invitation Settings

These settings let you customize the experience for your guests.

### Confetti on Wish

When this is turned on, a confetti burst appears on screen after a guest submits a wish. It makes the moment feel more fun and celebratory!

- **Emoji 1 / Emoji 2** — Choose which emojis appear in the confetti. The default is 🙏 and ❤️, but you can change them to anything you like. These fields only appear when Confetti on Wish is enabled.

### Sunday as Ahad

When this is turned on, Sunday is displayed as **"Ahad"** on your invitation — the Arabic/Islamic name for Sunday. This is a common choice for invitations following Islamic tradition.

### Wish Auto-Reply

Write a thank-you message (up to 300 characters) that guests automatically see after they submit a wish. For example:

> _"Thank you so much for your warm wishes! We're grateful you joined us on this special day. 🙏"_

Leave this blank and no auto-reply will appear.

### Sunday as Ahad

When turned on, Sunday is displayed as **"Ahad"** on the invitation (the Arabic/Islamic name for Sunday). This is a common preference for invitations following Islamic tradition.

## Saving Your Invitation

When you're ready, click **"Save"**.

| Status        | What it means                                                          |
| ------------- | ---------------------------------------------------------------------- |
| **Draft**     | Only you can see the invitation. Safe to edit without guests noticing. |
| **Published** | Your invitation is live. Anyone with the link can see it.              |

> **Tip:** Start with Draft, make sure everything looks right using the Preview button, then switch to Published when you're ready to share.
