# Media Library

The Media Library is where you manage audio files that can be used as background music on your invitation.

Go to **Dashboard → Media** to open it.

---

## System Audio vs. Your Uploads

There are two types of audio available:

| Type             | Description                                                                                                             |
| ---------------- | ----------------------------------------------------------------------------------------------------------------------- |
| **System Audio** | Tracks we've included in the app. Available to all users. You can listen to them but cannot delete or edit them.        |
| **My Uploads**   | Audio files you've uploaded yourself. Only you can see and use them. You can play a preview or delete them at any time. |

You don't have to upload anything. System Audio tracks are ready to use for free. But if you have a special song in mind, you can upload it here.

---

## How to Upload an Audio File

1. Go to **Dashboard → Media**.
2. Click **"Upload Audio"**.
3. Select your audio file from your device.
4. Wait for the upload to finish — the file will appear in your **My Uploads** list.

::: warning File Requirements

- Accepted formats: **MP3, WAV, OGG**
- Maximum file size: **5 MB** per file
  :::

---

## Managing Your Uploaded Files

| Action      | How to do it                                                    |
| ----------- | --------------------------------------------------------------- |
| **Preview** | Click the play button next to the file to listen to it.         |
| **Delete**  | Click the delete (trash) button to remove the file permanently. |

::: danger
Deleting an uploaded audio file is permanent. If the file is currently being used by an invitation, the music will stop working on that invitation.
:::

---

## Using Your Music on an Invitation

Once you've uploaded a file, here's how to use it:

1. Go to **Dashboard → Invitations** and click **Edit** on the invitation.
2. Scroll to **Section 6 — Background Music**.
3. Under the music source, select **"My Uploads"**.
4. Choose your uploaded file.
5. Optionally turn on **Autoplay** and **Loop**.
6. Save the invitation.
