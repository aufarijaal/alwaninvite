# Guests & Wishes

This is where you see every guest who has responded to your invitation, track RSVPs, and read the wishes they've left for you.

Go to **Dashboard → Guests & Wishes** from the sidebar.

---

## At-a-Glance Stats

At the top of the page you'll see a quick summary:

| Stat                  | What it shows                                                    |
| --------------------- | ---------------------------------------------------------------- |
| **Total Guests**      | Total number of guest entries (pre-added + those who responded). |
| **Attending**         | How many guests confirmed they are coming.                       |
| **Not Attending**     | How many guests said they can't make it.                         |
| **Maybe**             | Guests who responded but weren't sure yet.                       |
| **Pending**           | Guests who have been pre-added but have not responded yet.       |
| **Total Guest Count** | Sum of the "guests bringing" numbers across all RSVP responses.  |

---

## Attendance Statuses

Each guest's RSVP shows one of four statuses:

| Status            | Meaning                                                                    |
| ----------------- | -------------------------------------------------------------------------- |
| **Pending**       | The guest was pre-added but hasn't opened the invitation or responded yet. |
| **Attending**     | The guest confirmed they will be there.                                    |
| **Maybe**         | The guest opened the invitation and responded, but is unsure.              |
| **Not Attending** | The guest confirmed they won't be able to come.                            |

## Adding Guests in Advance

You can pre-add your guest list **before** the wedding. This is useful so you can create personalized invitation links that already include each guest's name.

::: info
You can add up to **500 guests** per invitation.
:::

There are two ways to add guests:

### Paste a List

Type or paste all guest names into a text box — one name per line. Great when you have a list ready in a spreadsheet or document.

```
Budi Santoso
Ani Rahayu
Pak Hasan
```

### Add One by One

Add guests individually using the input field. Useful for small groups or when you want to double-check each name.

### Duplicate Detection

When adding guests, the system automatically checks for duplicates. If a name already exists in your guest list, or appears more than once in your batch, it will be flagged before saving. You can review and remove the duplicates before confirming.

## WhatsApp Message Generator

Next to each guest in the table there are two copy buttons:

- **Copy WhatsApp Message** — Copies a ready-to-send WhatsApp message to your clipboard, including a warm greeting, a summary of the event(s), and the personalized invitation link. If you've set a custom WA template for that invitation, that template is used; otherwise the default message is sent.
- **Copy Link** — Copies just the personalized invitation link (e.g. `alwaninvite.my.id/invite/your-slug?to=GuestName`) to your clipboard, without any surrounding message text.

Just open WhatsApp, paste the message in the guest's chat, and send. The invitation link already includes their name, so when they open it they see a personal greeting.

---

## WhatsApp Message Template

You can customise the default WhatsApp invitation message for each invitation. Click the **"Configure WA Template"** button (the settings icon in the page header) to open the template editor.

**Available variables:**

| Variable  | Replaced with                    |
| --------- | -------------------------------- |
| `[guest]` | The guest's name                 |
| `[link]`  | The personalized invitation link |

The editor includes:

- A **live preview** showing exactly what the message will look like for a sample guest.
- **Prebuilt templates** in both Indonesian and English (formal, semi-formal, and casual styles) that you can apply with one click.
- An **Insert Variable** button to add `[guest]` or `[link]` at the cursor position.
- A **Reset to Default** button to restore the original template.

The custom template is saved per invitation. If you have multiple invitations, each can have its own template.

## Filtering, Searching & Sorting

Use the filters at the top of the guest table to narrow down the list:

| Filter                   | What it does                                                                           |
| ------------------------ | -------------------------------------------------------------------------------------- |
| **Search by name**       | Find a specific guest by typing their name (also searches wish messages).              |
| **Filter by status**     | Show only guests with a specific attendance status (Attending, Not Attending, etc.).   |
| **Filter by invitation** | If you have multiple invitations, filter guests by which invitation they responded to. |

You can also **sort** the table by clicking any column header (Date, Guest Name, Attendance, or Guest Count). Click the header again to reverse the sort order.

---

## Export to CSV

Click **"Export"** to download all your guest and RSVP data as a spreadsheet file (.csv).

The exported file includes:

- Date of response
- Guest name
- Attendance status
- Number of guests they're bringing
- Their wish message
- Which invitation they responded to

The file is named `wishes-[today's date].csv`. You can open it in Excel, Google Sheets, or any spreadsheet app. Great for printing a seating chart or sharing with your wedding organizer.

---

## Bulk Delete

You can select multiple guests at once and delete them all at the same time. Use the checkboxes on each row, then click **"Delete Selected"**.

::: warning
Bulk deleting guest records is permanent and cannot be undone.
:::
