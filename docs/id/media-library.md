# Perpustakaan Media

Perpustakaan Media adalah tempat kamu mengelola file audio yang bisa digunakan sebagai musik latar di undanganmu.

Buka dari **Dashboard → Media**.

---

## Audio Sistem vs. Unggahan Saya

Ada dua jenis audio yang tersedia:

| Jenis             | Keterangan                                                                                                                           |
| ----------------- | ------------------------------------------------------------------------------------------------------------------------------------ |
| **Audio Sistem**  | Lagu-lagu yang sudah kami sediakan di aplikasi. Tersedia untuk semua pengguna. Bisa didengarkan tapi tidak bisa dihapus atau diedit. |
| **Unggahan Saya** | File audio yang kamu unggah sendiri. Hanya kamu yang bisa melihat dan menggunakannya. Bisa di-preview atau dihapus kapan saja.       |

Kamu tidak harus mengunggah apapun. Lagu-lagu Audio Sistem siap digunakan secara gratis. Tapi jika kamu punya lagu khusus, bisa diunggah di sini.

---

## Cara Mengunggah File Audio

1. Buka **Dashboard → Media**.
2. Klik **"Unggah Audio"**.
3. Pilih file audio dari perangkatmu.
4. Isi **Judul** lagu (wajib). Judul otomatis terisi dari nama file, tapi bisa diubah.
5. Opsional: isi nama **Artis**.
6. Klik **"Unggah"** dan tunggu hingga proses selesai — file akan muncul di daftar **Unggahan Saya**.

::: warning Persyaratan File

- Format yang diterima: **MP3, WAV, OGG**
- Ukuran file maksimal: **5 MB** per file
- Judul **wajib** diisi
  :::

---

## Mengelola File yang Diunggah

| Tindakan      | Caranya                                                                      |
| ------------- | ---------------------------------------------------------------------------- |
| **Pratinjau** | Klik tombol putar di samping file untuk mendengarkannya.                     |
| **Hapus**     | Klik tombol hapus (ikon tempat sampah) untuk menghapus file secara permanen. |

::: danger
Menghapus file audio yang diunggah bersifat permanen. Jika file tersebut sedang digunakan oleh sebuah undangan, musik di undangan itu akan berhenti bekerja.
:::

---

## Menggunakan Musikmu di Undangan

Setelah mengunggah file, ini cara menggunakannya:

1. Buka **Dashboard → Undangan** dan klik **Edit** pada undangan yang diinginkan.
2. Gulir ke **Bagian 6 — Musik Latar**.
3. Di pilihan sumber musik, pilih **"Unggahan Saya"**.
4. Pilih file yang sudah diunggah.
5. Opsional: aktifkan **Putar Otomatis** dan **Ulangi**.
6. Simpan undangan.
