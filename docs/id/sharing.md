# Berbagi Undangan

Setelah undanganmu dipublikasikan, kamu siap untuk membagikannya kepada tamu!

---

## Link Undanganmu

Setiap undangan memiliki link unik berdasarkan URL slug yang kamu pilih saat membuat undangan. Tampilannya seperti ini:

```
alwaninvite.my.id/invite/slug-undanganmu
```

Kamu bisa menemukan link ini:

- Di kartu undangan di halaman **Daftar Undangan**.
- Dengan mengklik tombol **Pratinjau** — URL di browser adalah link yang bisa kamu bagikan.

---

## ✨ Link Personal — Menambahkan Nama Tamu

Kamu bisa membuat setiap undangan terasa personal dengan menambahkan nama tamu ke link. Saat tamu membuka link personal, undangan menyapa mereka dengan nama mereka — misalnya, _"Kepada Budi, kamu diundang ke pernikahan kami…"_

Caranya, tambahkan `?to=NamaTamu` di akhir link undangan:

```
alwaninvite.my.id/invite/slug-undanganmu?to=Budi
```

| Contoh                                 | Yang dilihat tamu                           |
| -------------------------------------- | ------------------------------------------- |
| `.../invite/budi-dan-ani`              | Sapaan umum (tanpa nama).                   |
| `.../invite/budi-dan-ani?to=Pak+Hasan` | _"Kepada Pak Hasan"_ muncul di undangan. 🎉 |

::: tip
Jika nama tamu mengandung spasi, ganti setiap spasi dengan `+`. Contoh: `?to=Pak+Hasan`.
:::

## Mengirim via WhatsApp

WhatsApp adalah cara paling populer untuk berbagi undangan pernikahan di Indonesia. Berikut beberapa tips:

- **Gunakan Generator Pesan WhatsApp** — Buka [Tamu & Ucapan](./guests-wishes), tambahkan tamu, dan gunakan tombol bawaan "Salin Pesan WhatsApp". Pesan sudah otomatis mencantumkan nama dan link personal tamu.
- **Kirim sebagai pesan, bukan hanya link saja** — Tambahkan sapaan hangat sebelum menempel link.
- **Coba link sendiri terlebih dahulu** — Buka link sekali sebelum mengirim, untuk memastikan tampilannya sudah benar di ponsel.
- **Bagikan linknya, bukan screenshot** — Link langsung memungkinkan tamu RSVP dan meninggalkan ucapan.

---

## Pratinjau Media Sosial

Saat kamu membagikan link undangan di WhatsApp, Instagram, atau Facebook, sebuah **kartu pratinjau** otomatis muncul — menampilkan nama pasangan dan gambar yang dibuat otomatis. Ini terjadi secara otomatis dan kamu tidak perlu melakukan apapun tambahan.

::: tip
Pastikan kamu sudah mengisi nama pasangan di undangan agar kartu pratinjau tampak bagus saat dibagikan!
:::

---

## ✅ Sebelum Berbagi — Daftar Periksa

Lewati daftar ini sebelum mengirim undangan ke tamu:

- [ ] Undangan sudah diatur ke status **Dipublikasikan** (bukan Draf).
- [ ] Nama pasangan sudah ditulis dengan benar.
- [ ] Detail acara (tanggal, waktu, lokasi) sudah benar.
- [ ] Sudah dipratinjau di layar ponsel.
- [ ] Link undangan berfungsi saat dibuka di browser.
- [ ] URL slug sudah sesuai keinginan (sulit mengubahnya nanti tanpa memutus link lama).
- [ ] Musik latar diputar dengan benar (jika menambahkan musik).
- [ ] Informasi hadiah sudah benar.
